# Randy Ramírez Ampie — Abogado Laboralista

Sitio web profesional de **Randy Ramírez Ampie**, abogado incorporado al Colegio de
Abogados y Abogadas de Costa Rica, con énfasis en **Derecho Laboral**.

## Contenido
- Perfil profesional
- Áreas de práctica (Laboral, Civil, Administrativo)
- Razones para elegirlo
- Trayectoria y formación
- Contacto (WhatsApp, teléfono, correo y ubicación)

## Tecnología
Sitio estático: HTML, CSS y JavaScript puro. Sin dependencias ni build.

## Estructura
```
index.html    Página principal
styles.css    Estilos
script.js     Interacciones (menú, animaciones)
```

## Desarrollo local
Abra `index.html` en el navegador, o sirva la carpeta:
```bash
python -m http.server 8000
```
