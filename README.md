# Abogado Laboralista — Sitio web profesional

Sitio web profesional de un **abogado** incorporado al Colegio de Abogados y Abogadas
de Costa Rica, con énfasis en **Derecho Laboral**.

> ⚠️ Versión **pseudo-anónima**: por ahora se omiten el nombre, los datos de contacto
> y métricas personales. Se reemplazan por etiquetas genéricas y placeholders.

## Contenido
- Perfil profesional
- Áreas de práctica (Laboral, Civil, Administrativo)
- Razones para elegirlo
- Trayectoria y formación
- Contacto (placeholders "a solicitud")

## Tecnología
Sitio estático: HTML, CSS y JavaScript puro. Sin dependencias ni build.
Incluye toggle de idioma **ES / EN**.

## Estructura
```
index.html    Página principal
styles.css    Estilos
script.js     Interacciones (idioma, menú, animaciones)
```

## Desarrollo local
Abra `index.html` en el navegador, o sirva la carpeta:
```bash
python -m http.server 8000
```
